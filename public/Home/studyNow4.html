Bài 3: Các lệnh cơ bản trong Git

Trong bài học này, chúng ta sẽ tìm hiểu về các lệnh cơ bản trong Git:

Các lệnh cơ bản

  • git init: Khởi tạo một repository Git mới.
  • git add: Thêm file vào staging area.
  • git commit: Lưu thay đổi vào repository.
  • git status: Kiểm tra trạng thái của repository.
  • git log: Xem lịch sử commit.
