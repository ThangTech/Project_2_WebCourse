Bài 3: Cú pháp cơ bản và cách chạy JavaScript

Trong bài học này, chúng ta sẽ tìm hiểu về cú pháp cơ bản của JavaScript và các cách để chạy mã JavaScript:

Cách thêm JavaScript vào trang web

Có ba cách chính để thêm JavaScript vào trang web:

1. Inline JavaScript

Thêm trực tiếp vào thuộc tính sự kiện của thẻ HTML:

<button onclick="alert('Xin chào!');">Nhấn vào đây</button>

2. Internal JavaScript

Sử dụng thẻ <script> trong trang HTML:

<!DOCTYPE html>
<html>
<head>
  <title>Trang web của tôi</title>
</head>
<body>
  <h1>Xin chào!</h1>
  
  <script>
    // Mã JavaScript ở đây
    console.log("Xin chào từ JavaScript!");
    document.write("Văn bản được tạo bởi JavaScript");
  </script>
</body>
</html>

3. External JavaScript

Tạo file JavaScript riêng và liên kết vào trang HTML:

<!-- Trong file HTML -->
<script src="script.js"></script>

// Trong file script.js
console.log("Mã JavaScript từ file bên ngoài");
alert("Xin chào từ file script.js!");

Cú pháp cơ bản của JavaScript

Câu lệnh và dấu chấm phẩy

Mỗi câu lệnh JavaScript thường kết thúc bằng dấu chấm phẩy (;). Mặc dù JavaScript có thể tự động thêm dấu chấm phẩy trong nhiều trường hợp, nhưng việc thêm dấu chấm phẩy là một thói quen tốt:

let x = 5;           // Khai báo biến x và gán giá trị 5
console.log(x);      // In giá trị của x ra console
alert("Xin chào!");  // Hiển thị hộp thoại thông báo

Chú thích (Comments)

JavaScript hỗ trợ hai loại chú thích:

// Đây là chú thích một dòng

/* Đây là chú thích
 nhiều dòng */

Phân biệt chữ hoa chữ thường

JavaScript phân biệt chữ hoa và chữ thường. Điều này có nghĩa là các biến, hàm và từ khóa phải được viết chính xác:

let name = "John";  // Đúng
let Name = "Jane";  // Biến khác, không phải 'name'
let NAME = "Jim";   // Biến khác nữa

// Các từ khóa phải viết đúng
let myVariable = 10;  // Đúng
Let myOtherVar = 20;  // Sai! 'Let' phải viết thường: 'let'

Công cụ để chạy và kiểm tra JavaScript

Console trong trình duyệt

Cách đơn giản nhất để thử nghiệm JavaScript là sử dụng console của trình duyệt:

  1. Mở trình duyệt (Chrome, Firefox, Edge, v.v.)
  2. Nhấn F12 hoặc chuột phải vào trang web và chọn "Inspect" (Kiểm tra)
  3. Chuyển đến tab "Console"
  4. Nhập mã JavaScript và nhấn Enter để thực thi

Môi trường phát triển trực tuyến

Có nhiều trang web cho phép bạn viết và chạy JavaScript trực tuyến:

  • JSFiddle (jsfiddle.net)
  • CodePen (codepen.io)
  • JS Bin (jsbin.com)

Node.js

Nếu bạn đã cài đặt Node.js, bạn có thể chạy JavaScript từ dòng lệnh:

// Lưu mã vào file example.js
console.log("Hello from Node.js!");

// Chạy từ terminal/command prompt
// node example.js
