Bài 3: Cú pháp cơ bản và cách chạy Python

Trong bài học này, chúng ta sẽ tìm hiểu về cú pháp cơ bản của Python và các cách để chạy mã Python:

Cách chạy Python

Có ba cách chính để chạy Python:

1. Sử dụng Interactive Shell

Chạy Python trực tiếp trong terminal hoặc command prompt: 

>> print("Hello, Python!")

2. Sử dụng file .py

Viết mã Python trong file và chạy bằng lệnh:

# File: hello.py
print("Hello, Python!")

3. Sử dụng IDE

Sử dụng các IDE như PyCharm, VS Code để viết và chạy mã Python.
